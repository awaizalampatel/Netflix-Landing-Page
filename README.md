# Netflix Landing Page

A professional, responsive landing page clone inspired by Netflix's official website. This project demonstrates modern web development techniques with a focus on user experience, performance, and accessibility.

## What This Project Does

This is a visually stunning landing page that looks and feels like the real Netflix website. It features:

- **Beautiful Hero Section** - A full-screen hero with a background image and call-to-action form
- **Interactive Navigation** - A header that changes appearance as you scroll, with a profile dropdown menu
- **Feature Showcases** - Four compelling sections highlighting Netflix's key features:
  - Watch on your TV
  - Download shows to watch offline
  - Watch on multiple devices
  - Create profiles for kids
- **FAQ Accordion** - An expandable question-and-answer section
- **Email Signup Forms** - Two places where visitors can enter their email to get started
- **Professional Footer** - Complete with links and contact information

## Key Features

### Visual Design
- Netflix's signature red and black color scheme
- Smooth animations and transitions
- Professional typography using Inter and Bebas Neue fonts
- Responsive design that works on all screen sizes

### Interactive Elements
- Scroll-triggered header background change
- Expandable FAQ accordion with smooth animations
- Profile dropdown menu on hover
- Email form submission with loading and success states
- Fade-in animations as you scroll down the page

### Technical Highlights
- Clean, semantic HTML5 structure
- Modern CSS with custom properties (variables)
- Vanilla JavaScript with no external dependencies
- Mobile-first responsive design
- Accessible markup with ARIA labels
- Optimized for performance

## How to Use

### Quick Start
1. Download or clone this repository
2. Open the `index.html` file in any modern web browser
3. That's it! No server or build tools required

### Viewing the Project
Simply double-click the `index.html` file, or right-click and choose "Open with" your preferred browser. The page will load immediately with all features working.

## Customization

You can easily customize this project:

### Colors
Change the Netflix red color by modifying the CSS variable:
```css
--netflix-red: #E50914;  /* Change this to any color */
```

### Content
- Edit text directly in `index.html`
- Replace images by updating the `src` attributes
- Add or remove FAQ items in the FAQ section

### Styling
- Modify `css/style.css` to change appearance
- All animations can be adjusted or disabled
